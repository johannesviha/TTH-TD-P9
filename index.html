Project 1: Mobile-First Form

A responsive employee management web app that fetches and displays random users from the Random User Generator API. Users' images, names, emails, and cities are shown in a grid. Clicking on a user opens a modal window with more details, including cell number, address, and birthdate. Built with HTML, CSS, and JavaScript using the fetch API.

Project 2: Dynamic Image Gallery

Create a responsive image gallery with search functionality. Built using HTML, CSS Grid, and JavaScript. Features thumbnail images, modal lightbox with details, and real-time image filtering. Ensured cross-browser compatibility.

Project 3: Dashboard UI

Design a responsive dashboard UI using HTML, CSS Grid, and JavaScript. Includes header with icons, navigation, chart widgets, and user-related sections. Utilized chart.js for dynamic data visualization. Achieved a consistent look across different screen sizes.

Project 4: Responsive Contact Form

Crafted a mobile-first contact form following a user-friendly design. Utilized HTML and CSS with a Google Font for typography. Structured form sections for "Contact Information" and "Newsletter." Integrated form field types like text, email, telephone, select menu, checkboxes, radio buttons, and textarea. Ensured cross-browser compatibility and responsive layout for tablet and desktop screens. Added :focus states for input fields and validated code with HTML and CSS validators.
